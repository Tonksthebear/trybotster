ChainParams
Parameters for controlling the behavior of PQR key chains. It’s recommended to use the Default API for overriding values, as future values may be added to this struct, and Default allows them to be added in a backwards-compatible fashion. IE: let params = ChainParams{max_jump: 10, ..Default::default()};
